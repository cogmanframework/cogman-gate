# Cogman Kernel

**⚠️ This is a reference/placeholder directory in the public repository.**

## Kernel Source Code

The C++ kernel source code is maintained in a **separate repository** for IP protection:

- **Repository:** https://github.com/cogmanframework/cogman-kernel
- **Access:** Private/Protected
- **Purpose:** Core formulas (CORE-1 to CORE-9) implementation

## For Users

### Installation

Kernel binaries are distributed via the main installation script:

```bash
# Install Cogman Gate (includes kernel binary)
curl -fsSL https://raw.githubusercontent.com/cogmanframework/cogman-gate/main/install.sh | bash
```

### For Developers

If you need access to kernel source code:
1. Contact the maintainers for repository access
2. Or use the pre-compiled binaries from releases
